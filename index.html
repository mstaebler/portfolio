Vote Poll

Fullstack voting web app. I learned a lot on this project.

    My takeways were:
  • Don't always add a depenency.
    • I spent more time debugging weird interactions between various react packages than actually writing the code.
  • Will this code add value?
    • I aborted converting my code over to redux due to unnecessary complexity for no real gain due to the small scope of my project.

Created with: React, NodeJs and ExpressJs MongoDB
